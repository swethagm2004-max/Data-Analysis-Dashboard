Project Title

Financial Data Analysis and Interactive Dashboard Using Power BI

Project Objectives

To analyze financial transaction data and identify key trends and patterns.

To visualize important financial metrics such as total transactions, loan amounts, EMI, and balances.

To compare financial performance across account types and branches.

To support data-driven decision-making using interactive dashboards in Power BI.
